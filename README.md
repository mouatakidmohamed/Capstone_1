# Capstone_1 - Sales Analysis Project

## Project Overview
This project is a SQL-based sales analysis designed to explore and evaluate sales performance data. The goal is to identify trends, summarize revenue, and gain insights that support business decision-making.

## Sales Territory
The sales territory being analyzed in this project is **Maryland**.

## Objective
The main objectives of this analysis are:
- Examine monthly revenue trends within the Maryland territory
- Analyze store performance based on location data
- Summarize and aggregate sales data using SQL queries
- Identify patterns that can help improve business insights

## Tools Used
- MySQL
- SQL scripting
- Excel (PivotChart)
- Git & GitHub
